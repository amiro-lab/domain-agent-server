전체 정리 파이프라인
정규화 → 중복 머지를 한 번에 실행. 정규화로 표기 변종을 통일한 뒤 그 결과 위에서 의미 중복을 잡아내므로 따로 도는 것보다 정확합니다. retag-untagged(LLM 호출 비용)는 이 파이프라인에 포함하지 않음.
태그 정규화 (canonical: 소문자 + 하이픈)
V5 → v5, batch_api → batch-api, Domain:Auth → domain:auth 같은 표기 변종을 통일합니다. PROTECTED 태그(v*_fixed)는 그대로 보존. 신규 저장은 자동 정규화되므로 이 작업은 옛 누적분 백필용.
중복 머지 (Jaccard ≥ threshold)
비슷한 description의 메모리 클러스터를 canonical 1개에 합칩니다. confidence는 max, tags는 union으로 결합하고 나머지는 archive 됩니다 (90일 후 영구 삭제). PROTECTED 태그(v*_fixed)는 자동 제외.
